# Leadminer

Leadminer is a NodeJS app to extract and qualify email addresses from your own mailboxes.

## Installation

Use [app](https://pip.pypa.io/en/stable/) to install leadminer.

```bash
command line installation
```

## Usage

```bash

$ usage
$ usage

```
